# Installation

The DeepSparse Platform contains 4 libraries (DeepSparse Community, DeepSparse Enterprise, SparseML, and SparseZoo).

Each package is installed with [PyPI](https://pypi.org/user/neuralmagic/). It is recommended to install in 
a [virtual environment](https://docs.python.org/3/library/venv.html) to encapsulate your local environment.

## Quick Start

Run the following commands which installs standard setups for the DeepSparse Platform.

```bash
pip install deepsparse[server] sparseml[torch,torchvision]
```
